-----------------------------------------
Arguments:
-----------------------------------------

args[0] = number of arguments of the frameworks F, G
args[1] = number of arguments of the frameworks H
================= optional ==============
args[2] = number of threads				[Default: 6]
args[3] = name of the experiment			[Default: "SearchStrongUC"]
args[4] = Path to the folder, in which a new folder will be created for the started experiment. If this argument is empty, then the folder will be created in the same place the program was started in

Generator F:	Enumerating
Generator G: 	Enumerating
Generator H:	Enumerating

Enumerating Generators generate all frameworks possible for the specified number of arguments.
